========================
✅ Multi-modloader support (NeoForge, Fabric, Forge)
✅ Automatic mod management (Ferium integration)
✅ Configurable update frequency (1-24 hours, default 4)
✅ Configurable weekly updates (day + hour selection)
✅ Full-featured admin dashboard (port 8001)
✅ HTTP mod distribution server (port 8000)
✅ RCON server control integration

✅ Live activity logging
✅ Headless operation (no GUI needed)
✅ Systemd service support

✅ Security features (password auth, path validation)
✅ Real-time status updates
✅ Mod add/remove/upgrade operations
✅ Configuration management UI

Deployment Methods Supported
=============================
1. Standalone (manual): ./neorunner_env/bin/python3 run.py run
2. Systemd service: sudo systemctl start neorunner
3. Tmux session: tmux new-session -s minecraft -d
4. Docker (prepared): Dockerfile ready for containerization
5. Ansible (template): Playbook structure provided
