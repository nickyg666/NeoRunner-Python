System Installation

Debian/Ubuntu Installation

$ sudo apt-get update
$ sudo apt-get install -y git python3 python3-pip python3-venv
$ python3 -m venv mod-editing-env
$ source mod-editing-env/bin/activate && pip install gitpython flask

Fedora Installation

$ sudo dnf update
$ sudo dnf install -y git python3 python3-pip python3-virtualenv
$ python3 -m venv mod-editing-env
$ source mod-editing-env/bin/activate && pip install gitpython flask

Amazon Linux Installation

$ sudo yum update
$ sudo yum install -y git python3 python3-pip python3-venv
$ python3 -m venv mod-editing-env
$ source mod-editing-env/bin/activate && pip install gitpython flask

Arch Linux Installation

$ sudo pacman -Syu
$ sudo pacman -S --noconfirm git python python-pip python-virtualenv
$ python -m venv mod-editing-env
$ source mod-editing-env/bin/activate && pip install gitpython flask
